Manual mount via JS API

You can also mount programmatically by targeting an existing element: 

<div id="corsa-stream-widget"></div>
<script src="bundle.js"></script>
<script>
  CorsaEmbed.mount({
    elementId: "corsa-stream-widget",
    username: "luke",
    streamId: "abc123"
  });
</script>
